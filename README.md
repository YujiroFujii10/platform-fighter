# Platform Fighter

This is a 2D platform fighting game made in Godot 4.  
It features:

- Various stages
- local multiplayer
- keyboard controls
- Hitbox-based combat system
- Menu and transitions


## Player Controls

PLAYER 1


Left: A

Right: D

Up: W

Down: S

Attack: F

Special: G

Shield: R

Light: hold shift

PLAYER 2


Left: p

Right: ]

Up: -

Down: [

Attack: o

Special: i

Shield: 9

Light: hold enter


## Controls

Movement:

Walk: light + left/right

Run: left/right

Dash: Tap left/right

Crouch: down

Jump / Full Hop: up

Jump / Short Hop: tap up

Fast Fall: down (in air)


Attacks:

Tilt Attacks / Jab: light + attack + left/right/up/down/none

Smash Attacks: attack + left/right/up/down (simultaneously)

Smash Attack Charge: attack + left/right/up/down (simultaneously and hold)

Dash Attack: attack + left/right

Specials: special + left/right/up/down/none

Aerials (in air): attack + left/right/up/down/none


Defense:

Shield: shield

Roll: shield + left/right

Spot Dodge: shield + down

Air Dodge / Directional Air Dodge: shield + left/right/up/down/none (in air)


Ledge Options:

Regular Getup: left/right

Jump Getup: up

Ledge Roll: shield

Ledge Drop: down


## About

This project is a full-featured platform fighter showcasing:

Combat mechanics: Hitboxes, tilts, smashes, specials, aerials, dash attacks
Movement & physics: Walk, run, dash, jumps, short hops, fast falls
Defensive systems: Shielding, rolls, spot dodges, air dodges
Game structure: Multiple stages, menus, scene transitions, and character selection
Code architecture: Object oriented and reusable state machine for character actions
Animation & visuals: Smooth movement and attack animations
